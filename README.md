## ✨ Features

* **🤖 AI Travel Planner:** Powered by Gemini AI, instantly generate custom travel itineraries and save them directly to the cloud.
* **💳 Smart Wallet:** A beautifully designed dashboard that pulls live destination images (Unsplash) and real-time weather data (OpenWeather) for all your upcoming trips.
* **🔐 Biometric Vault:** Military-grade local security. Lock your passport, visa, and private travel documents behind native Fingerprint and FaceID authentication.
* **☁️ Multi-User Cloud Sync:** Built on Supabase Authentication and Postgres, ensuring your travel data is securely synced and isolated to your specific account.
* **🗣️ AI Translator:** Real-time text translation and pronunciation assistance to help you navigate foreign locations with ease.
* **🎨 Premium UI/UX:** Built with a dark-themed aesthetic featuring glassmorphism, dynamic Hero animations, and satisfying haptic feedback.

---

## 🛠 Tech Stack

* **Frontend:** [Flutter](https://flutter.dev/) (Dart)
* **Backend & Auth:** [Supabase](https://supabase.com/)
* **AI Engine:** Google Gemini API
* **Live Data:** OpenWeather API, Unsplash API
* **Security:** `local_auth` (Native Android/iOS Biometrics)
